- **01_Main_Project_Prompt.txt** - פרומפט ראשי
- **02_Flask_Application.txt** - פרומפט Flask
- **03_Frontend_Development.txt** - פרומפט Frontend

3. התחל עם קובץ 01
4. המשך בסדר עד קובץ 06
