# 📧 Outlook Email Manager - מערכת ניהול מיילים חכמה

מערכת ניהול מיילים מתקדמת עם אינטגרציה ל-Outlook, ניתוח AI חכם, ומערכת למידה מותאמת אישית.

## 🌟 תכונות עיקריות

### 📧 ניהול מיילים
- **חיבור ישיר ל-Outlook** - גישה למיילים אמיתיים
- **ניתוח חשיבות חכם** - זיהוי אוטומטי של מיילים חשובים
- **קטגוריזציה אוטומטית** - סיווג מיילים לפי סוג
- **סיכום אוטומטי** - סיכום תוכן המיילים
- **חילוץ פעולות** - זיהוי משימות נדרשות

### 🤖 בינה מלאכותית
- **אינטגרציה עם Gemini AI** - ניתוח מתקדם של תוכן
- **למידה מותאמת אישית** - המערכת לומדת מהעדפות המשתמש
- **משוב משתמש** - שיפור מתמיד של הניתוח
- **פרופיל משתמש** - שמירת דפוסי עבודה

### 🎨 ממשק משתמש
- **עיצוב מודרני** - ממשק אינטואיטיבי וידידותי
- **תמיכה בעברית** - ממשק מלא בעברית
- **סטטיסטיקות** - נתונים על המיילים והשימוש
- **ניהול העדפות** - התאמה אישית של המערכת

## 🚀 התקנה והרצה

### דרישות מערכת
- **Windows 10/11**
- **Python 3.6+**
- **Microsoft Outlook** (אופציונלי)
- **API Key של Google Gemini** (אופציונלי)

### התקנה מהירה
1. **הורד את הפרויקט**
2. **פתח PowerShell כמנהל**
3. **נווט לספריית הפרויקט:**
   ```powershell
   cd c:\Users\ronni\outlook_email_manager
   ```
4. **הרץ את הסקריפט:**
   ```powershell
   .\quick_start.ps1
   ```

### הרצה מהירה
```powershell
.\quick_start.ps1
```

### התקנה ידנית
1. **התקן תלויות:**
   ```bash
   pip install -r requirements.txt
   ```
2. **הרץ את השרת:**
   ```bash
   python app_with_ai.py
   ```
3. **פתח דפדפן וגש ל:**
   ```
   http://localhost:5000
   ```

## 📊 API Endpoints

### מיילים
- `GET /api/emails` - קבלת כל המיילים
- `GET /api/important-emails` - קבלת מיילים חשובים
- `GET /api/stats` - סטטיסטיקות

### AI ולמידה
- `GET /api/ai-status` - סטטוס AI
- `POST /api/toggle-ai` - הפעלה/כיבוי AI
- `GET /api/learning-stats` - סטטיסטיקות למידה
- `POST /api/toggle-learning` - הפעלה/כיבוי למידה

### העדפות
- `GET /api/preferences` - קבלת העדפות
- `POST /api/preferences` - שמירת העדפות
- `GET /api/user-preferences` - העדפות מתקדמות
- `POST /api/user-preferences` - שמירת העדפות מתקדמות

### משוב
- `POST /api/user-feedback` - רישום משוב משתמש
- `GET /api/user-profile` - פרופיל משתמש
- `POST /api/reset-learning` - איפוס למידה
